```jsx title="Create URL"
https://reurl.eu/api/v2/create?url=https://yourawesomewebsite.com
```

## Get URL clicks
```jsx title="Get url clicks"
https://reurl.eu/api/v2/clicks?shortUrl=6MHADVSEx
```
